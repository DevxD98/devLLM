# `research/paper_notes/`

One note per paper, written **in our own words**. The goal is not a summary you could get
from an abstract — it is a record of *what we understood* and *what we borrowed*.

## Convention

- Filename = snake_case of the paper title: `attention_is_all_you_need.md`
- Every note follows the same shape (see any existing note as the template):
  `Why we read it → Core idea in plain words → Mechanisms we reuse →
  What we ignore (and why) → Open questions → One-line takeaway`.
