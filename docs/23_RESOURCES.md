# 23 — Curated Resources

> **Prerequisites:** [`22_FUTURE_VERSIONS.md`](22_FUTURE_VERSIONS.md).

---

## Purpose

A curated catalog of high-quality courses, repositories, visual guides, and reference implementations that complement JimmyLabs's educational journey.

---

## Primary Courses & Video Series

- **Andrej Karpathy — *Neural Networks: Zero to Hero***
  - *Description:* The gold standard video series for building neural networks, autograd, tokenizers, and GPTs from scratch in Python.
  - *Relevance:* Spiritual foundation for JimmyLabs's Stage 2–3 learning path ([`docs/03_LEARNING_PATH.md`](03_LEARNING_PATH.md)).

- **3Blue1Brown — *Neural Networks* & *Essence of Linear Algebra***
  - *Description:* Geometric visual intuitions for matrix transformations, vectors, dot products, and backpropagation.
  - *Relevance:* Visual companion for [`docs/04_MATHEMATICS.md`](04_MATHEMATICS.md).

---

## Reference Repositories

- **nanoGPT (Karpathy):** Minimal, readable PyTorch implementation of GPT-2 training/inference (~300 lines).
- **minGPT (Karpathy):** Clean, educational re-implementation of GPT.
- **micrograd (Karpathy):** Tiny scalar-valued autograd engine demonstrating backpropagation.
- **MLX (Apple):** Efficient machine learning framework designed specifically for Apple Silicon unified memory.

---

## Interactive Visual Guides

- **Jay Alammar — *The Illustrated Transformer***
  - Excellent visual breakdown of attention matrices, encoder-decoder blocks, and token vectors.
- **bronka — *LLM Visualization***
  - Interactive 3D visualization of tensor flows through a transformer model.
